## Vercel 佈署

1. 登入 Vercel
2. 點 `Add New` → `Project`
3. 選擇 GitHub repo：`budafang/ai_shot`
4. `Framework Preset` 選 `Other`
5. `Build Command` 留空
6. `Output Directory` 留空
7. 按 `Deploy`

此專案為純靜態 HTML/CSS/JS 網站，不需要額外 build。
